# BNGU_EXAM

本项目是为 BNGU 招新考核准备的综合演示仓库。项目旨在通过 ROS2 的不同通信机制复现考核内容，并展示过往的人脸识别学习成果，以此体现对加入 BNGU 的决心与技术能力。

## 📂 项目结构

本项目包含三个主要的工作空间（Workspace），分别对应不同的考核内容与展示：

*   **`exam1_ws`**：存放基础考核内容。
*   **`exam2_ws`**：通过 **ROS2 服务（Service）通信** 的方式复现考核内容。
*   **`exam3_ws`**：存放过往学习的人脸识别项目。
*   **`videos`**：存放用于演示的视频素材（`video1`, `video2`, `video3`）。

人脸识别部分（`exam3_ws`）需要在虚拟环境下运行。项目中已包含配置好的虚拟环境文件夹 `face_env`，请在运行前激活该环境。

## 🚀 运行示例

项目所需的演示视频素材均存放在 `videos` 目录下，分别对应各项考核的展示内容。

1.  **视频素材路径**：
    *   `videos/video1`
    *   `videos/video2`
    *   `videos/video3`

2.  **运行流程**：
    *   请根据各 `ws` 目录下的具体说明，结合上述视频素材进行功能演示。
